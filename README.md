# 迅雷NAS助手

通过浏览器扩展实现右键将链接发送到NAS上的迅雷客户端。

## 功能特性

- 右键菜单快速发送下载链接到NAS迅雷
- 自动化操作迅雷NAS客户端界面
- 支持批量任务添加
- 首次使用配置向导

## 安装方法

### 手动安装

1. 下载本项目源代码
2. 打开Chrome/Edge浏览器，进入扩展管理页面 (`chrome://extensions/`)
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目文件夹

### 注意

由于未通过官方商店分发，浏览器会显示"未列在商店"的警告，这是正常现象。

## 首次使用配置

**重要：第一次使用前必须完成配置**

1. 点击浏览器工具栏中的扩展图标，打开配置界面
2. 输入你的NAS迅雷完整访问地址，例如：
   - `http://192.168.5.2:2345/webman/3rdparty/pan-xunlei-com/index.cgi/#/home`
3. 点击保存配置
4. 配置完成后即可正常使用右键菜单功能

## 使用说明

1. 完成首次配置后，确保NAS上的迅雷客户端已启动
2. 在任意下载链接上右键
3. 选择"发送到NAS迅雷"菜单项
4. 扩展将自动打开迅雷界面并添加下载任务

## 技术实现

基于浏览器扩展API，通过在迅雷下载界面注入脚本模拟用户点击操作，与迅雷NAS客户端界面交互。

## 常见问题

**Q: 为什么右键菜单无法使用？**  
A: 请确保已完成首次配置，正确设置了NAS迅雷的访问地址。

**Q: 如何修改NAS地址？**  
A: 点击扩展图标重新打开配置界面，修改地址后保存。

**Q: 支持的迅雷NAS版本？**  
A: 支持迅雷NAS版及类似Web界面的版本。

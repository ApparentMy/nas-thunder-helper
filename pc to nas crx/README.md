# NAS迅雷助手

一个用于自动化操作迅雷NAS版客户端的浏览器扩展。

## 使用须知

本插件通过模拟用户点击与迅雷NAS客户端交互，旨在提升操作效率。请注意：

- 本工具为独立第三方开发，与迅雷官方无关。
- 使用本工具可能违反《迅雷软件许可协议》，相关风险由用户自行承担。
- 严禁使用本工具进行任何侵犯版权的下载活动。开发者不对用户行为造成的后果负责。

## 安装说明

1. 下载插件源代码
2. 在浏览器中打开扩展管理页面 (`chrome://extensions/`)
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"，选择插件目录

## 使用方法

1. 确保迅雷NAS客户端已开启
2. 在下载链接上使用插件功能
3. 插件将自动操作迅雷界面添加任务

## 常见问题

**账号被限制怎么办？**
请阅读迅雷用户协议，自动化操作可能违反条款。请合理使用。

**插件安全吗？**
代码完全开源，仅在与迅雷页面交互时操作，不收集任何用户数据。

**插件失效了？**
迅雷客户端更新可能导致界面变化，需要更新插件适配。
